### 디스트럭처링 할당 = 구조분해할당

구조화된 배열과 같은 이터러블, 객체를 비구조화 시켜 1개 이상의 변수에 개별적으로 할당하는 것을 말한다. 배열과 같은 데이터에서 필요한 값만 추출하여 변수에 할당하여 사용할 때 유용하다. 

### 배열 구조분해할당

- 배열 구조분해할당의 오른쪽은 리터러블이어야 하며, 왼쪽은 할당 받을 변수를 선언해야 한다.
- 할당 기준은 배열의 인덱스 순서대로 할당된다.

```jsx
const arr = [1, 2, 3];

// ES6 배열 디스트럭처링 할당
// 변수 one, two, three를 선언하고 배열 arr을 디스트럭처링하여 할당한다.
// 이때 할당 기준은 배열의 인덱스다.
const [one, two, three] = arr;

console.log(one, two, three); // 1 2 3
```

### 객체 구조분해할당

- 객체 구조분해할당의 오른쪽은 객체이어야 하며, 왼쪽은 할당받을 변수를 선언해야 한다.
- 할당 기준은 객체의 프로퍼티 키가 된다. 객체 구조분해할당에서는 변수의 순서는 상관없으며 선언된 변수 이름과 프로퍼티 키가 일치하면 할당된다.

```jsx
const user = { firstName: 'Ungmo', lastName: 'Lee' };

// ES6 객체 디스트럭처링 할당
// 변수 lastName, firstName을 선언하고 user 객체를 디스트럭처링하여 할당한다.
// 이때 프로퍼티 키를 기준으로 디스트럭처링 할당이 이루어진다. 순서는 의미가 없다.
const { lastName, firstName } = user;

console.log(firstName, lastName); // Ungmo Lee
```
